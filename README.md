# Panda-PyBullet

Panda simulation with PyBullet

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

The project has been developed and tested on **Ubuntu 18.04** and **Python 3.7.1**.

### Installation

All the required dependencies are listed in the *requirements.txt* file.
```
pip3 install -r requirements.txt
pip3 install -e .
```

### Run the simulation

See the 'Getting started' notebook

## Built With

* [PyBullet](https://pybullet.org/wordpress/) - The simulator used for the project.
